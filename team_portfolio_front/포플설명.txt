로그인 페이지
아이디, 비밀번호 유효성 검사(공백)

회원가입 페이지
회원가입 입력 정보들 유효성 검사(공백, 조건)

공통 헤더
검색바 클릭시 검색창 나옵니다.
검색창에서 검색하기 클릭시 검색 페이지로 이동합니다.
헤더 우측에 있는 가운데 아이콘을 누르면 마이 페이지로 이동합니다.
헤더 우측에 있는 마지막 아이콘을 누르면 로그인 페이지로 이동합니다.

메인 페이지(반응형)
모임 이미지를 호버시 이미지가 확대 됩니다.
모임 이미지의 하트를 누르면 하트 색상이 채워집니다.
모임의 이미지를 클릭시 상세 페이지로 이동합니다.

상세 페이지(반응형)
작은 사진들을 클릭시 크게 나옵니다.
우측에 정보창이 스크롤 할 때 마다 따라다닙니다.
위시 리스트를 누르면 하트 색상이 채워집니다.
별점 주기 버튼을 클릭시 드래그 형식으로 별점을 주는 창이 나옵니다.

검색 페이지(반응형)
모임 이미지를 호버시 이미지가 확대 됩니다.
모임 이미지의 하트를 누르면 하트 색상이 채워집니다.
모임의 이미지를 클릭시 상세 페이지로 이동합니다.

마이 페이지(반응형)
리밋 만들기 클릭시 모임을 만들 수 있는 팝업이 나옵니다(공백 유효성 검사).
가입한 리밋 관리 클릭시 가입한 리밋 목록들이 나옵니다
개설한 리밋 관리 클릭시 개설한 리밋 목롣들이 나옵니다
정보 수정하기 클릭시 정보를 수정할 수 있는 팝업이 나옵니다(비밀번호체크 유효성 검사)
